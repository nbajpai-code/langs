## Introduction to Assamese
Assamese (অসমীয়া, Oxomiya) is an Eastern Indo-Aryan language spoken mainly in the Indian state of Assam, where it is an official language. It is the easternmost indigenous Indo-European language and acts as a lingua franca for the wider region of Northeast India.

## Alphabet & Pronunciation
Assamese is written using the Assamese alphabet, which is closely related to the Bengali script with a few minor differences (e.g., the letter for 'ra' and 'wa').
- Features a unique set of alveolar stops instead of the dental/retroflex distinction found in most other Indic languages.
- Distinctly lacks retroflex consonants in its phonology.
- Has an 'x' sound (a voiceless velar fricative, similar to the 'ch' in German "Bach" or Scottish "loch").

## Grammar Overview
- **Word Order:** Subject-Object-Verb (SOV).
- **Agglutinative traits:** Uses postpositions and suffixes to indicate case and function.
- **Classifiers:** Widely uses numeral classifiers when counting objects (e.g., 'jon' for people, 'khon' for flat objects).
- **Verbs:** Conjugated based on person and tense but *not* gender.
